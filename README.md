# ReactionCLI

A Python-based command-line reaction time tester featuring false start detection, performance statistics, and personal record tracking.

Built and tested on **macOS**.

---

## Features

- Reaction time measurement in milliseconds
- Randomized start delay to prevent anticipation
- False start detection
- Performance statistics
- PR tracking

---

## Requirements

- Python 3.x
- macOS (uses `afplay` for audio playback)

---

## Run

```bash
python3 rxnCLI.py
```
